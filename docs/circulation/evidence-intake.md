# Evidence Intake Loop
*The disciplined entry point of the circulatory layer.*

INTEVIA accepts new evidence only through a clear, intentional pathway.
This loop ensures that all incoming evidence:

- is captured  
- is validated  
- is placed in the Evidence Spine  
- is routed to the correct chamber  
- strengthens coherence rather than diluting it  

The intake loop consists of:

1. Submission  
2. Review  
3. Placement  
4. Circulation  
5. Integration  

Each step exists to protect the organism from noise
and to ensure that only meaningful, aligned evidence
enters its memory and structure.
