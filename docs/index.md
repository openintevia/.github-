# INTEVIA — Cross‑Chamber Index
*The coherence map of the organism.*

INTEVIA is formed from five differentiated chambers.
Each chamber holds a distinct function.
Integration emerges from their relationships.

## forge — the chamber of making
Inputs:
- architectural intent
- lineage decisions
- build tasks

Outputs:
- artefacts
- prototypes
- implementation logs

Feeds:
→ organism (structure)
→ docs (evidence)
→ www (public artefacts)

## organism — the chamber of being
Inputs:
- conceptual models
- structural definitions
- coherence patterns

Outputs:
- system architecture
- conceptual lineage
- internal logic

Feeds:
→ hats (operating rules)
→ docs (explanations)
→ www (public articulation)

## hats — the chamber of operating
Inputs:
- Human–AI interaction patterns
- governance principles
- triad choreography

Outputs:
- operating model
- interaction discipline
- execution posture

Feeds:
→ forge (workflow)
→ organism (alignment)
→ docs (methodology)

## www — the chamber of presence
Inputs:
- public‑ready artefacts
- identity surfaces
- threshold signals

Outputs:
- public membrane
- external visibility
- first contact surfaces

Feeds:
→ docs (references)
→ organism (identity)
→ forge (artefacts)

## docs — the chamber of articulation
Inputs:
- knowledge
- structure
- lineage
- decisions

Outputs:
- documentation spine
- conceptual clarity
- communicable architecture

Feeds:
→ all chambers
→ the world

The organism is forming.
