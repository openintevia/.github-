# Integrative Loop
*The organism’s cycle for resolving cross-layer contradictions.*

Integration is not static.
As the organism evolves, tensions arise between layers.
The integrative loop enables INTEVIA to:

- detect cross-layer contradictions  
- revisit the relevant structures or behaviours  
- evaluate the source of misalignment  
- resolve tensions through adjustment  
- integrate updated understanding across the organism  

The loop consists of:

1. Detection  
2. Examination  
3. Adjustment  
4. Integration  
5. Re-alignment  

This is how the organism maintains unity in motion.
